# 📄 Resume Analyzer using NLP

A Natural Language Processing (NLP) application that analyzes resumes and matches them with job descriptions using multiple text processing and similarity techniques.

---

# 🚀 Features

✅ Resume Parsing (.docx and .pdf)

✅ NLP Preprocessing
- Tokenization
- Cleaning
- Stopword Removal
- Lemmatization

✅ Skill Extraction

✅ Text Representation Methods
- TF-IDF
- Bag of Words (BoW)

✅ Cosine Similarity Matching

✅ Resume-Job Matching Score

✅ Data Visualization

✅ Streamlit Web Application

✅ Dark Theme UI

---

# 🧠 NLP Concepts Used

This project applies several NLP concepts:

- Natural Language Processing
- Text Cleaning
- Tokenization
- Lemmatization
- Stopwords Removal
- TF-IDF Vectorization
- Bag of Words Representation
- Cosine Similarity
