# Resume Analyzer using NLP

This project analyzes resumes and matches them with job descriptions using Natural Language Processing techniques.

## Features

- Resume parsing (.docx and .pdf)
- NLP preprocessing
- Skill extraction
- TF-IDF similarity
- Resume-job matching
- Streamlit web interface
- Data visualization

## Technologies Used

- Python
- NLTK
- Scikit-learn
- Streamlit
- Matplotlib
- PDFPlumber

## Run Project

```bash
streamlit run app.py
